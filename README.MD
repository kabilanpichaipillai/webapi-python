# FastAPI User Management App

A simple **FastAPI-based REST API** for managing users, built with:
- Python & FastAPI
- CRUD operations
- Proper HTTP status codes & error handling
- Dockerized deployment
- Automated testing using **TDD (pytest)**

---

## Features

- Welcome endpoint
- Create, Read, Update, Delete (CRUD) users
- Input validation using Pydantic
- In-memory database (demo purpose)
- Swagger UI & ReDoc
- Automated API tests (TDD)
- Docker support

## Run the application locally

uvicorn main:app --reload

Application will be available at:

API: http://localhost:8000

Welcome API: http://localhost:8000/welcome

Get User: http://localhost:8000/user

Swagger UI: http://localhost:8000/docs

---

## API Endpoints

### Welcome
GET /welcome

### Get default user
GET /user

### Get user by ID
GET /user/{user_id}

### Create user
POST /user

### Update user
PUT /user/{user_id}

### Delete user
DELETE /user/{user_id}

----

## Automated Testing (TDD)

### Run test locally
pytest -v

## Docker Support

### Build Docker Image
docker build -t fastapi-user-app .

docker run -d -p 8000:8000 --name fastapi-container fastapi-user-app
